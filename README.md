# Tienda Virtual

Aplicación interactiva de tienda virtual desarrollada con HTML, CSS y JavaScript.  
Incluye un catálogo dinámico de productos y un carrito de compras totalmente funcional.

## Enlace al proyecto
https://erbusaa06.github.io/tienda-virtual/

## Organización del proyecto
- index.html → Página principal
- styles.css → Estilos de la tienda
- script.js → Lógica del catálogo y carrito
- README.md → Información del proyecto

## Funcionalidades
- Catálogo dinámico de productos
- Agregar productos al carrito
- Eliminar productos del carrito
- Límite de compra de 100 €
- Vaciar carrito
- Interfaz moderna y responsive

## Cómo usar
1. Selecciona un producto y pulsa "Agregar".
2. Consulta el carrito en la parte derecha.
3. Haz clic en un producto del carrito para eliminarlo.
4. Pulsa "Vaciar carrito" para reiniciar la compra.
